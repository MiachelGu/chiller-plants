Logs

GET /api/v1/logs/<site-id>
Name Description
API Version v1
API to interact with timeseries data of chiller plants
Query Parameter Optional Description
site-id Yes Chiller plant site ID
start Yes UTC Date in format %Y-%m-%d. Data logged after this date will be returned
end Yes UTC Date in format %Y-%m-%d. Data logged on or before this date will be returned
freq No Sampling rate of the data. Accepted values: years, months, days, hours, minutes
fields Yes Comma separated list of chiller plant data fields to return (lowercase)
order No Order of the returned data. 1 (ascending), -1(descending). The order of comma separated field will be implicitly used as precendence.
func No Aggregation function applied on the data. Accepted values: avg (default), max, min, sum
limit No Maximum number of results to return. At most 200.
token No Next page token. Token to query data in the next page 
Response Format
          {
            "apiVersion": "v1",
            "queryParams": {
              # query parameters sent as request
            },
            "nextPageToken": # value, 
            "results": [
              {
                "_id": # UTC timestamp,
                "field": # value,
              }
            ]
          }
